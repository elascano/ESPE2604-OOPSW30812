# SoftwareSecureStore

## Project Description
**SoftwareSecureStore** is a comprehensive management system designed for minimarkets. It streamlines the administration of inventory, stock control, and customer relations. This project focuses on data persistence and secure storage using advanced file handling in Java.

## Key Modules
* **Inventory & Stock:** Real-time monitoring of warehouse products and stock levels.
* **Receiving (Almacén/Recepción):** Management of incoming goods and supplier delivery tracking.
* **Customer Management:** Database to handle customer information and sales history.
* **Data Storage:** Implementation of CSV and GSON (JSON) files for secure and organized data handling.

## Technical Stack
* **Language:** Java
* **Architecture:** Object-Oriented Programming (OOP)
* **Libraries:** GSON for JSON processing
* **Environment:** NetBeans IDE
