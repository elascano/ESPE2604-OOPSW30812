# ShoppingCenter - Python

Proyecto convertido de Java a Python 3.

## Estructura
```
ShoppingCenter_Python/
├── shopping_center.py    ← Archivo principal (equivalente al main)
└── customer.py           ← Clase Customer
```

## Cómo ejecutar

### Con terminal:
```bash
python shopping_center.py
```

### Con NetBeans:
1. Abrir NetBeans
2. File → New Project → Python → Python Project
3. Copiar los archivos en el proyecto
4. Click derecho en shopping_center.py → Run File

## Requisitos
- Python 3.7 o superior
- NetBeans con plugin de Python (nbPython) instalado
